Docker 搭建

编辑

  1. 虚拟机里面安装kali系统后，需要查看版本信息：
    uname -a （查看系统信息）
    uname -v （查看内核版本）

  2. 更换apt的源：

    ​ 查看：cat /etc/apt/sources.list

    ​ 修改：输入 sudo vim /etc/apt/sources.list 然后输入密码 然后按回车，再按i进入编辑模式，

    ​ 开始编辑：

    选择源，推荐阿里云源，有些已经不能用了

    deb http://mirrors.aliyun.com/kali kali-rolling main non-free contrib

    deb-src http://mirrors.aliyun.com/kali kali-rolling main non-free contrib

    ​ 编辑好后，按ESC键退出，然按照提示后输入：（：wq）保存退出。强制退出（：wq!）

  3. 切换到root用户
    sudo passwd root //进行root密码修改
    su root// 切换root用户

  4. ​编辑
